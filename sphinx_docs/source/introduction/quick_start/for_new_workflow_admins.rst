2.3 For New Workflow Admins: Configure a simple Workflow
==========================================================

This guide is intended for new workflow administrators who want to configure workflows.

1. Login to LoonFlow
2. Click the "Workflow" button in the left sidebar
3. Click the "New Workflow" button
  .. figure:: ../../images/new_workflow.png
    :width: 100%
    :align: center
    :alt: Workflow List

4. Fill in the workflow basic information
  .. figure:: ../../images/fill_workflow_basic_info.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Basic Info

5. Finish form design
  .. figure:: ../../images/fill_workflow_form_design.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Form Design

6. Finish process design
  .. figure:: ../../images/fill_workflow_process_design01.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Process Design

  .. figure:: ../../images/fill_workflow_process_design02.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Process Design

  .. figure:: ../../images/fill_workflow_process_design03.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Process Design

5. Click the "RELEASE" button, then set the workflow version
  .. figure:: ../../images/release_workflow.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Version
6. After workflow is released, the user can start to use the workflow
  .. figure:: ../../images/use_the_new_workflow.png
    :width: 100%
    :align: center
    :alt: Use the New Workflow
