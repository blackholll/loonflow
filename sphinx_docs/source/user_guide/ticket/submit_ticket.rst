submit ticket
===============


This guide is intended for users who want to submit a ticket.

1. Login to LoonFlow
2. Clieck "Workbench" in the left sidebar
3. Select a workflow from the dropdown menu

   .. figure:: ../../images/select_workflow.png
      :width: 100%
      :align: center
      :alt: Select Workflow

4. Click the "New Ticket" button
5. Fill in the ticket information

   .. figure:: ../../images/fill_ticket_form.png
      :width: 100%
      :align: center
      :alt: Fill Ticket Info

6. Click corresponding button listed in the bottom of the dialogpage to submit the ticket.
7. The ticket will be submitted and the next handle user will receive a notification.
