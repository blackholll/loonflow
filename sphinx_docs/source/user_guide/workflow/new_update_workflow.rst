new and update workflow
========================

This guide is intended for new workflow administrators who want to configure workflows.

1. Login to LoonFlow
2. Click the "Workflow" button in the left sidebar
3. Click the "New Workflow" button
  .. figure:: ../../images/new_workflow.png
    :width: 100%
    :align: center
    :alt: Workflow List

4. Fill in the workflow basic information
  .. figure:: ../../images/fill_workflow_basic_info.png
    :width: 100%
    :align: center
    :alt: Fill Workflow Basic Info

5. Finish form design

    :width: 100%
    :align: center
    :alt: Fill Workflow Form Design
